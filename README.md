# Config File Validator — C#

A lightweight command-line tool that validates JSON configuration files for missing fields, invalid URLs, and incorrect data types.

Perfect for troubleshooting configuration issues during API integrations or software setup.

---

## Features
✔ Validates required fields  
✔ Verifies URL format  
✔ Ensures correct data types (int, string, etc.)  
✔ Produces clean, readable error messages  
✔ Extremely fast (runs in milliseconds)

---

```bash
dotnet run sample_config.json
